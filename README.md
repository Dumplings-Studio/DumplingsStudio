# Dumpling Studio — Website

A minimal static site for Dumpling Studio (games portfolio / prototypes).

How to run locally

- Using Python 3 (quick):

```bash
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.

- Using Node.js (http-server):

```bash
npm install -g http-server
http-server -c-1
```

Files

- [index.html](index.html) — main page
- [styles.css](styles.css) — styling
- [app.js](app.js) — small JS to render game cards
- [logo.svg](logo.svg) — simple logo
